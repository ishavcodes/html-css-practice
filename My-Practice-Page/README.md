# Dev.Practice — My First HTML & CSS Webpage

A practice webpage built from scratch using only HTML and CSS — no JavaScript, no frameworks.

---

## 🌐 Live Preview

> Open `index.html` in your browser to view the page.

---

## 📁 Project Structure

```
project/
├── index.html       → Main HTML file
├── style.css        → All styling
└── images/          → Icons used in cards and button
    ├── construction.png
    ├── palette.png

    └── rocket.png
```

---

## 📄 What's on the Page

| Section | Description |
|---------|-------------|
| **Navbar** | Logo + navigation links with hover effect |
| **Hero** | Heading, subtext, and explore button |
| **About** | Short description of what the page covers |
| **Cards** | 3 cards explaining core concepts |
| **Contact Form** | Name, email, message inputs + send button |
| **Footer** | Simple copyright line |

---

## 🧠 Concepts Practiced

- **HTML Structure** — semantic tags, nesting, forms
- **CSS Styling** — colors, fonts, spacing, shadows, transitions
- **Flexbox** — navbar layout, card row, form alignment
- **CSS Variables** — reusable color tokens via `:root`
- **Hover Effects** — buttons and cards with smooth transitions
- **Google Fonts** — Source Serif 4
- **Font Awesome** — icon in the explore button
